# Vectorly App

The front end web app for Vectorly. Built with Typescript and React!

### Requirements

- Node.js > v12
- Yarn

### Installation
```
$ git clone git@github.com:lemonJS/vectorly.git
$ yarn install
$ yarn dev
``` 

### Running the dev server
```
$ yarn dev
```

### Exporting the app as a static site
```
$ yarn build
$ yarn export
```

### Running the typechecker
```
$ yarn run tsc
```
